# **App Name**: LearnVerse

## Core Features:

- User Authentication: Secure user registration and login using email/password or Google OAuth.
- Course Generation: Generate a course from user input using AI, including SFW check, syllabus creation, content curation, and lecture notes generation using an LLM tool.
- Syllabus display: Clean display of modules with title, YouTube videos, and lecture notes.
- Progress Tracking: Save user course generation details and module completion status to track overall progress.
- Interactive Quizzes: Generate multiple-choice quizzes based on lecture notes using an AI LLM tool for user assessment.
- PWA Installability: Enable 'Add to Home Screen' functionality via service worker and manifest for an app-like experience.
- Offline Access: Cache course syllabus, lecture notes, and quizzes using a service worker for offline learning.

## Style Guidelines:

- Primary color: A deep blue (#3F51B5) to evoke trust, stability, and knowledge.
- Background color: White (#FFFFFF) to ensure readability and a clean interface.
- Accent color: A vibrant purple (#8E44AD) for interactive elements and calls to action.
- Headline font: 'Space Grotesk' sans-serif for titles; Body font: 'Inter' sans-serif for readability.
- Code font: 'Source Code Pro' for displaying code snippets in lessons.
- Use consistent, outline-style icons for navigation and module status.
- Subtle animations to acknowledge user interactions, such as marking a module as complete.
